## 一、信息架构 (Information Architecture)

### 1.1 全局导航

采用 **底部 TabBar，4个固定 Tab**：

```
┌─────────────────────────────────────────┐
│                  内容区                  │
│                                         │
│                                         │
├──────────┬──────────┬──────────┬────────┤
│  总览     │  账本    │  投资    │  信用  │
│ Overview │ Ledger  │ Invest  │ Credit │
└──────────┴──────────┴──────────┴────────┘
```

| Tab | 图标 (SF Symbol) | 说明 |
|-----|----------------|------|
| 总览 Overview | `chart.pie.fill` | 净资产全貌 + 各模块摘要 |
| 账本 Ledger | `list.bullet.rectangle.fill` | 日常收支记录与统计 |
| 投资 Invest | `chart.line.uptrend.xyaxis` | 持仓汇总与行情 |
| 信用 Credit | `person.2.fill` | 借贷往来管理 |

**全局悬浮按钮 (FAB)**：位于 TabBar 上方中央，点击弹出快速记账 Sheet，在所有 Tab 均可见。图标：`plus.circle.fill`，主题色填充。

**设置入口**：总览页右上角 `gear` 图标，进入独立 Settings 页面。

## 三、图表设计规范

### 3.1 图表类型与使用场景

| 图表 | 组件 | 使用场景 | 空状态 |
|------|------|---------|-------|
| 饼图 / 甜甜圈图 | Swift Charts `SectorMark` | 分类占比、资产配置 | 显示"暂无数据"占位圆 |
| 柱状图 | Swift Charts `BarMark` | 每日/周收支对比 | 空柱 + 引导文案 |
| 折线图 | Swift Charts `LineMark` | 净值趋势、资产变化 | 直线（¥0）+ 引导文案 |
| 排行列表 | 自定义 `ProgressView` 行 | 分类消费排行 | "本期无消费记录" |

### 3.2 颜色规范

| 语义 | 颜色 | 用途 |
|------|------|------|
| 支出 Expense | `.systemRed` | 支出金额、支出柱 |
| 收入 Income | `.systemGreen` | 收入金额、收入柱 |
| 盈利 Profit | `.systemGreen` | 投资浮盈 |
| 亏损 Loss | `.systemRed` | 投资浮亏 |
| 债权 Credit | `.systemGreen` | 净债权（外债 > 我欠） |
| 债务 Debt | `.systemOrange` | 净债务（我欠 > 外债） |
| 主题色 | `.systemBlue` | FAB、主按钮、Tab 选中 |
| 警告 Warning | `.systemOrange` | 价格过期、即将到期 |

颜色不作为唯一信息传达手段，配合图标和文字标签一起使用。

### 3.3 空状态 (Empty State) 规范

每个带有列表或图表的页面需定义空状态：

| 页面 | 空状态插图建议 | 文案 | CTA |
|------|-------------|------|-----|
| 账本列表 | `tray` SF Symbol | "还没有记录，开始记第一笔吧" | "记一笔 +" |
| 账户管理 | `creditcard` SF Symbol | "添加你的账户并填写期初余额" | "添加账户 +" |
| 账本统计 | `chart.bar` SF Symbol | "本期暂无数据" | - |
| 投资持仓 | `chart.line.uptrend.xyaxis` SF Symbol | "添加你的第一个持仓" | "添加持仓 +" |
| 信用账本 | `person.2` SF Symbol | "暂无借贷记录" | "新建借贷 +" |

## 四、关键交互设计

### 4.1 快速记账流程

```
点击 FAB
  → QuickEntrySheet 从底部弹出（半屏）
  → 键盘自动弹出，聚焦金额输入
  → 输入金额 → 选择分类（最近3个常用 + 滚动更多） → 选择账户（默认上次使用）
  → 可填/不填名称
  → 点击"完成" → Sheet 关闭，交易列表更新（带短暂动画反馈）
  → 点击"更多详情" → 进入完整记账页（含日期/备注/账户/转账/凭证/币种）

长按 FAB
  → 弹出多模块快捷菜单：记支出 / 记收入 / 转账 / 记投资 / 记欠款 / 记别人欠我
  → 选择后跳转对应模块的快速录入表单
```

### 4.2 时间段选择器 (PeriodPicker)

- 胶囊形按钮组：`周 / 月 / 年 / 自定义`，点击即时切换。
- 月/周模式下左右箭头翻页（上一月/下一月），支持手势左右滑动切换。
- "自定义"弹出日期区间选择器 (`DateRangePicker`)，选择后 badge 显示日期范围。
- 选中的时间段状态持久化（下次打开同模块保留上次选择）。

### 4.3 投资价格刷新

- 手动刷新：右上角 `arrow.clockwise` 按钮，点击后触发所有持仓价格更新（v1.0 通过 API）。
- 刷新中显示 `ProgressView` in-place，成功后更新最后刷新时间。
- 价格超过 1 小时未更新时，持仓行以橙色警告色标注 `⚠ 价格已过期`。

### 4.4 借贷还款

- 详情页底部固定"记录还款"按钮。
- 弹出 Sheet，输入还款金额、日期、还款账户（可选，关联账户则同步余额）。
- 若剩余金额归零，自动将 `Debt.status` 改为 `.settled`，并弹出"🎉 已结清"提示。
- 分期借贷按每期金额生成到期提醒，支持按期循环提醒。

### 4.5 滑动操作

- 列表行向左滑动：显示"编辑"（蓝色）和"删除"（红色）。
- 删除需二次确认（Alert）。
- 信用账本向左滑动额外显示"结清"快捷操作。

## 五、设计风格

| 元素 | 规范 |
|------|------|
| 字体 | 系统字体 (SF Pro)，数字金额使用 `.monospacedDigit` |
| 图标 | SF Symbols 3.0+，填充 (fill) 变体为主 |
| 圆角 | 卡片 `cornerRadius: 16`，按钮 `cornerRadius: 12` |
| 卡片阴影 | 暗黑模式无阴影，浅色模式轻微 shadow（opacity 0.08） |
| 间距 | 水平边距 16pt，卡片内边距 16pt，列表行高 60pt |
| 主题色 | `.systemBlue` 作为 Accent Color |
| 布局 | `LazyVStack` + 分组卡片，避免嵌套滚动冲突 |
| 过渡动画 | `withAnimation(.spring(response: 0.3))` 用于列表变更 |
